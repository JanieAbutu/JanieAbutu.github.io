AI-Augmented SOC Alert Triage Pipeline

Problem

SOC environments generate high-volume alerts with inconsistent triage quality and heavy manual effort.

What I Built

An automated SOC triage pipeline that ingests alerts via webhook, enriches them with threat intelligence, maps activity to MITRE ATT&CK, and applies AI-assisted severity scoring for prioritization.

Detection Logic

  • IOC enrichment (VirusTotal API)
  • ATT&CK mapping for classification
  • Rule-based + AI hybrid scoring

Impact

  • Improved consistency of alert triage
  • Reduced manual investigation effort
  • Standardized incident prioritization logic

Stack

Python, n8n, PostgreSQL, Ollama, VirusTotal API
