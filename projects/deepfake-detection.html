Deepfake Image Detection using EfficientNet-B7

A deep learning-based forensic vision system built to detect AI-generated and manipulated facial images using transfer learning and EfficientNet-B7.

The Problem

AI-generated media has made manipulated images increasingly realistic and difficult to detect manually, creating risks in identity verification, fraud detection, and digital trust.

Traditional computer vision approaches struggle with:

  • Subtle manipulation artifacts
  • Rapidly evolving generative models
  • Poor generalization to unseen synthetic data

The core challenge was building a model that learns forensic signals rather than dataset-specific artifacts.

What I Built

A binary image classification system using EfficientNet-B7 and transfer learning to classify images as real or deepfake.

  • EfficientNet-B7 backbone (ImageNet pretrained)
  • Binary classification head
  • PyTorch training pipeline
  • Data preprocessing and augmentation
  • Checkpointing and early stopping
  • Evaluation metrics (precision, recall, F1, AUROC)

How It Works

  1. Load FaceForensics++ dataset frames.
  2. Apply resizing, normalization, and augmentation.
  3. Load pretrained EfficientNet-B7 model.
  4. Replace classification head with binary classifier.
  5. Fine-tune model on real vs deepfake images.
  6. Train using PyTorch with early stopping and checkpoints.
  7. Evaluate using accuracy, precision, recall, F1, AUROC.
  8. Generate ROC curves and confusion matrices.

Thought Process

The goal was not just accuracy, but generalization across unseen deepfake distributions.

  • Transfer learning over training from scratch
  • Strong augmentation to reduce overfitting
  • Balanced evaluation metrics over accuracy alone
  • Reproducible training pipeline design

The key question: can the model detect manipulation patterns beyond the dataset it was trained on?

Impact

  • 98%+ classification performance
  • Strong precision-recall balance
  • Robust generalization on test data
  • End-to-end reproducible ML pipeline

Demonstrates practical application of computer vision in cybersecurity-oriented forensic analysis and synthetic media detection.
