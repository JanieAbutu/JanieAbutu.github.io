Deepfake Image Detection using EfficientNet-B7

A deep learning-based forensic vision system designed to detect AI-generated and manipulated facial images using transfer learning and EfficientNet-B7.

The Problem

AI-generated media has made manipulated images increasingly realistic and difficult to detect manually. Deepfakes introduce risks across identity verification, fraud detection, journalism integrity, and digital trust systems.

Traditional computer vision approaches struggle with:

  • Subtle manipulation artifacts
  • Rapidly evolving generative models
  • Poor generalization to unseen synthetic data

The core challenge was building a model that learns forensic signals rather than dataset-specific shortcuts.

What I Built

I built a binary image classification system using EfficientNet-B7 and transfer learning to classify images as real or deepfake.

  • EfficientNet-B7 pretrained backbone (ImageNet)
  • Custom binary classification head
  • PyTorch training pipeline
  • Data preprocessing and augmentation pipeline
  • Model checkpointing and early stopping
  • Evaluation using multiple metrics

The system was trained on FaceForensics++ dataset frames and optimized for balanced detection performance.

How It Works

  1. Dataset images are loaded from FaceForensics++ frames.
  2. Images are resized, normalized, and augmented.
  3. EfficientNet-B7 pretrained on ImageNet is loaded.
  4. The classification head is replaced with a binary classifier.
  5. Model is fine-tuned on deepfake vs real image dataset.
  6. Training is optimized using early stopping and checkpoints.
  7. Performance is evaluated using accuracy, precision, recall, F1, and AUROC.
  8. ROC curves and confusion matrices are generated for analysis.

Thought Process

The key objective was not just classification accuracy, but generalization. Deepfake detection fails when models learn dataset artifacts instead of forensic patterns.

To address this, the design focused on:

  • Transfer learning instead of training from scratch
  • Strong augmentation strategies
  • Balanced evaluation metrics over accuracy alone
  • Reproducible training pipeline design

The guiding question was whether the model could detect manipulation patterns beyond the training dataset.

Impact

  • 98.07% classification accuracy on test data
  • High precision-recall balance across classes
  • Robust generalization on unseen validation samples
  • End-to-end reproducible ML pipeline

The project demonstrates practical application of computer vision in security-focused forensic analysis, with direct relevance to fraud detection and synthetic media verification systems.
