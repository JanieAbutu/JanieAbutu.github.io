AI-Augmented SOC Alert Triage Pipeline

Context

SOC environments struggle with alert fatigue due to high-volume, low-context security events.

Architecture

Webhook-based ingestion → enrichment layer → MITRE ATT&CK mapping → AI-assisted scoring → PostgreSQL logging.

Detection Engineering Design

Combined rule-based heuristics with LLM-assisted classification to improve alert prioritization consistency.

Implementation

  • Python-based ingestion and enrichment pipeline
  • VirusTotal threat intelligence integration
  • Ollama LLM for contextual analysis
  • PostgreSQL for structured audit logging

Impact

  • Improved triage consistency
  • Reduced manual investigation overhead

Stack

Python, n8n, PostgreSQL, VirusTotal API, Ollama
